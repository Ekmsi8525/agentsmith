## What is agentsmith?

Agentsmith helps you run artificial intelligence agents on your computer. Many AI programs require complex commands and technical knowledge. This software simplifies the process. It acts as a central hub for your AI tools. You can connect it to major models like Claude, Gemini, or Codex. 

The software uses specialized profiles to handle different tasks. These profiles teach the AI how to behave for specific jobs like writing code, organizing files, or analyzing data. You install one core program, pick a profile, and the system manages the rest. It removes the need to configure settings manually.

## ⚙️ System Requirements

Your computer needs specific parts to run this software well. Please check these requirements before you start the setup process:

*   Operating System: Windows 10 or Windows 11.
*   Memory: At least 8 gigabytes of RAM.
*   Storage: 500 megabytes of free space.
*   Internet: An active connection to download large files and connect to AI services.
*   Account: You need an active API key from an AI provider like Anthropic or Google to use the advanced features.

## 🛠️ Setting Up Your First Agent

When you open the program for the first time, it checks for updates and prepares the base environment. 

1.  Select a work-type profile from the list on the left side of the window. These profiles change how the AI interacts with your computer.
2.  Provide your API key when prompted. The software stores this key locally to communicate with AI services. 
3.  Click the Connect button.
4.  Test the connection by typing a simple instruction in the box at the bottom. For example, ask the agent to summarize a text file or help you draft a document.

The software displays status icons during this process. A green icon means everything is working. A red icon means the agent cannot reach the AI service. Recheck your API key if you see a red icon.

## 📋 Choosing a Profile

Profiles change what your agent can do. You can switch between these at any time:

*   Coding Assistant: This profile focuses on syntax, debugging, and software architecture. It connects directly to your code editors.
*   Task Automator: This profile manages repetitive work. It can rename files, move folders, or format documents.
*   Data Analyst: This profile looks for patterns in spreadsheets and documents. It provides summaries and report structures.
*   Creative Writer: This profile assists with brainstorming and editing. It maintains a specific tone based on your preference. 

## 🛡️ Keeping Things Secure

Security matters when you give an AI access to your computer. Agentsmith keeps your data on your machine. The internal settings never share your personal files with the outside world unless you explicitly ask the agent to upload them. 

The software does not record your screen. It only processes the information you provide in the chat window. Keep your API key private. Do not share your key file with anyone. Delete your API key from the settings if you decide to stop using the service.

## 🔧 Frequently Asked Questions

What if the program crashes?
Restart the application. If the problem continues, delete the temporary cache folder found in your user settings directory and run the installer again.

Can I run multiple agents at once?
Yes. You can open multiple windows of the program. Each window runs its own profile and manages its own tasks.

Will this slow down my computer?
The software uses a small amount of memory while idle. It uses more power while it generates text or processes files. If your computer feels slow, close other programs that use lots of memory.

Does this support local models?
This version relies on cloud-based models for better performance. Future updates might include options for small, local models.

How do I remove the software?
Open your Windows Control Panel. Select the option for installed programs. Find Agentsmith in the list and click Uninstall. This removes the core program and the desktop shortcut. 

## 📝 Troubleshooting Tips

If you encounter errors, look at the logs. The program generates a log file each time it runs. You can access this by clicking the Help menu and selecting View Logs. These logs contain technical details that might explain why a command failed.

Always ensure your internet connection remains stable during large tasks. If the internet cuts out, the agent stops working immediately. Restart the agent once your connection returns to resume your work.
